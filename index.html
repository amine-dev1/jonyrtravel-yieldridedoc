JonyTravel & YieldRide
API Reference

Guide complet d'intégration pour les partenaires B2B. Gérez les réservations, tarifications et excursions via des endpoints REST sécurisés.

REST / JSON B2B
🔐 Authentification — Headers obligatoires

Toutes les requêtes vers l'API doivent inclure les headers suivants. La clé API vous est fournie lors de l'onboarding partenaire.

HTTP Headers 
{
  "Content-Type": "application/json",
  "X-API-Key": "<VOTRE_API_KEY>"
}
⚠️
401 Unauthorized — Retourné si la clé API est manquante, invalide ou expirée.

2.4 — Annuler une réservation

/api/booking/cancel
Paramètres Query String
Paramètre Type Requis Description
reference string Oui Numéro de réservation à annuler
phone string Oui Téléphone du client (vérification)
reason string Non Raison de l'annulation
ℹ️
Exemple : DELETE /api/booking/cancel?reference=JT39458&phone=+212600112233&reason=Changement de plans
⚠️
Politique d'annulation : Annulation gratuite si la demande est faite plus de 12h avant la prise en charge. Une pénalité s'applique en cas d'annulation tardive.

3.4 — Informations d'annulation

/api/v1/cancellation-info
Paramètres Query String
ParamètreTypeRequisDescription
booking_idstringNonID de la réservation à consulter
referencestringOuiRéférence de la réservation à consulter
ℹ️
Exemple : GET /api/v1/cancellation-info?booking_id=12333333&reference=111
⚠️
Cet endpoint retourne les conditions d'annulation avant de procéder à l'annulation effective. Utilisez-le pour informer le client des pénalités éventuelles.
